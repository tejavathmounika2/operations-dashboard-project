# Operations Dashboard

## Project Overview

Operations Dashboard is a React-based dashboard that displays live operational data using public APIs. It provides real-time insights into finance, weather, market trends, and business operations.

## Features

- Live Bitcoin Price
- Live Ethereum Price
- USD to INR Exchange Rate
- Current Time
- Hyderabad Weather
- India GDP
- Technology News
- Bitcoin Trend Chart
- Employee Cards
- Monthly Sales Analytics
- Air Quality Index (AQI)
- Market Price Widget
- Gold Price Widget
- SOP Trigger Widget
- Auto Refresh
- Last Updated Timestamp
- Local Storage Caching

## Technologies Used

- React
- Vite
- JavaScript
- Axios
- Recharts
- CSS

## APIs Used

- CoinGecko API
- Frankfurter API
- Open-Meteo API
- World Bank API
- Hacker News API
